What can I do in Linux?

You can go about linux in a couple of ways.
The first is to just use it like you would a regular operating system! Modern linux is just as simple and intuative as Windows or Mac. With most linux distrobutions (or versions) even including a built in store!
The next and much more powerful way to use linux is through the terminal, a box that allows you to type in commands to get an output! While this can be confusing at first it can make you multitudes faster when using your PC.

Lets learn some commands!

cd

This command is used to change your directory.
ex) cd Pictures

ls

This command lists all files in your current directory.

mkdir

This command createsd a new diectory.
ex) mkdir myFolder

rmdir

Removes a directory

cat

Dumps file's contents into the terminal

cp

Copies a file to another file.
ex) cp file1.txt newFile.txt

mv

Similar to copy but deletes original file. Moving the file.

pwd

Prints out the directory you are currently in.

rm

Removes a file (but wont remove folders!)

touch

Creates a new empty file
ex) touch newFile.new

locate

This will locate a file on the system

find

Works just like locate but searches for folders too!

grep

Trims output to only show specific text
ex) grep hello file.txt
(finds all instances of hello in file.txt)

sudo

Runs a program in root, aka administrator.
ex) sudo cd /root
(only admins can go here!)

du

This tells you how much space a file takes up. Disk Usage
